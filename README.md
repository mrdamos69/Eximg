# Eximg Photo Editor

Пример кроссплатформенного приложения на Flutter для редактирования фотографий. Позволяет:

- выбирать фото из галереи или делать снимок камерой;
- открывать встроенный редактор с кадрированием, зеркалированием, поворотом и настройками;
- применять готовые фильтры, наклейки и текстовые элементы;
- сохранять отредактированный кадр в виде нового файла.

## Быстрый старт
1. Установите Flutter (iOS/Android) и добавьте его в `PATH`.
2. В корне проекта выполните:
   ```bash
   flutter pub get
   flutter run
   ```
3. На экране выберите снимок и откройте редактор для применения эффектов.

### Android
- Минимальная версия SDK: 26.
- Разрешения камеры и доступа к фото уже прописаны в `android/app/src/main/AndroidManifest.xml`.

### iOS
- Минимальная версия iOS: 13.0.
- Разрешения для камеры и фотобиблиотеки добавлены в `ios/Runner/Info.plist`.
- После установки зависимостей запустите `cd ios && pod install`, затем открывайте `Runner.xcworkspace` в Xcode или запускайте `flutter run` из корня проекта.
- Из-за запрета на бинарные файлы иконки не хранятся в репозитории: при сборке Xcode автоматически вызывает `tools/generate_app_icons.py`, а вручную их можно сгенерировать командой `python tools/generate_app_icons.py`.

Используются пакеты `image_picker` для выбора фото и `image_editor_plus` для инструментов редактирования.
